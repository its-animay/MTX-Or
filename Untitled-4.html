MTX-O algorithm is optimized in such a way that it shows result upto 99.98 % accuracy of data after collection of data we are using apriori and clustering algorithm to store data in it after processing and warehousing of data we use convolution Neural network- long short term memory and matrix based relay algorithm for better optimization and maximum accuracy
